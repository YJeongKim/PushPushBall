# Push Push Ball

<br>

### 게임 소개

- 기존 push push 게임을 응용하여 포켓볼을 접목시킨 공 넣기 게임
- 두 명의 Player가 제한된 시간동안 제한된 횟수 안에 공을 움직여 구멍에 넣는 게임
- 기존 push push 게임에서 Match, Dice, Timer, Rank 기능이 추가됨
